Create programs to group related change initiatives for coordinated management and reporting.

## Creating a new program

1. Navigate to **Programs** in the sidebar
2. Click **New Program**
3. Enter program details:

| Field | Description |
|-------|-------------|
| **Name** | Clear, descriptive title (e.g., "2026 Digital Transformation") |
| **Description** | Overview of program objectives, scope, and goals |

4. Click **Create**

## Program structure

After creation, your program provides:

### Overview dashboard

- Total initiatives in the program
- Combined stakeholder count
- Aggregate impact summary
- Overall timeline

### Initiative list

- All linked projects
- Individual status of each
- Quick navigation to project details

## Best practices

<AccordionGroup>
  <Accordion title="Use clear naming conventions">
    Include the year and theme: "2026 Finance Modernization" or "HR Systems Consolidation Program"
  </Accordion>
  <Accordion title="Define program scope clearly">
    Write a description that explains what belongs in this program and what doesn't.
  </Accordion>
  <Accordion title="Start with related initiatives">
    Group initiatives that share stakeholders, timelines, or strategic objectives.
  </Accordion>
  <Accordion title="Review program groupings periodically">
    As initiatives evolve, reassess whether program groupings still make sense.
  </Accordion>
</AccordionGroup>

## After creation

Once your program exists:

1. **Add initiatives** - Link existing projects or create new ones within the program
2. **Review stakeholder overlap** - Identify people affected by multiple initiatives
3. **Coordinate timelines** - Ensure initiatives don't overwhelm stakeholders simultaneously
4. **Set up program reporting** - Configure dashboards for leadership visibility

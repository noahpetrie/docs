## Integration benefits

### Stakeholder synchronization

Automatically import and update stakeholders from your identity provider:

- Keep stakeholder data current without manual updates
- Sync organizational hierarchies and reporting lines
- Import user photos and contact information

### Communication channels

Reach stakeholders through familiar platforms:

- Send survey invitations via Teams or Slack
- Post change updates to team channels
- Enable direct messaging for sensitive communications

### Workflow automation

Automate common tasks:

- Scheduled stakeholder syncs
- Automatic notifications for key events
- Channel monitoring for change-related discussions

## Managing integrations

Access integration settings:

1. Go to **Settings** in the sidebar
2. Click **Integrations**
3. View connected services and configuration options

### Connection status

Each integration shows:

- **Connected** - Active and working
- **Disconnected** - Not configured
- **Error** - Needs attention

### Permissions

Only organization admins can:

- Connect or disconnect integrations
- Modify integration settings
- View sync logs and history

Organization members can use connected integrations but cannot configure them.

## Data privacy

Integration data handling:

- Data synced from external systems is stored securely
- You control what data is synced via configuration
- Disconnecting an integration stops all data sync
- Previously synced data remains unless manually deleted

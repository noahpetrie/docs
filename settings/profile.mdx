title: Profile settings

---

Configure your personal information, display preferences, and regional settings.

## Accessing profile settings

1. Click **Settings** in the sidebar
2. Select **Profile** from the settings menu

## Profile picture

Upload a profile picture that appears throughout the platform:

1. Click **Upload Image**
2. Select an image file (PNG, JPEG, or GIF)
3. Maximum file size: 10MB

Your avatar appears in comments, activity logs, and team views.

## Personal information

Update your basic profile details:

| Field | Description |
|-------|-------------|
| **First name** | Your given name |
| **Last name** | Your family name |
| **Email** | Your primary email (read-only, set during signup) |
| **Job role** | Your position or title |
| **Company** | Your organization name |
| **Phone** | Contact phone number with country code |
| **Country** | Your location for regional defaults |
| **LinkedIn** | Link to your LinkedIn profile |

## Time preferences

Configure how dates and times display throughout the platform:

### Timezone

Select your local timezone. This affects:

- Scheduled communication times
- Survey deadlines
- Activity timestamps
- Report date ranges

### Time format

Choose between:

- **12-hour** - Displays as 2:30 PM
- **24-hour** - Displays as 14:30

### Date format

Select your preferred date display:

- **MM/DD/YYYY** - US format (03/15/2026)
- **DD/MM/YYYY** - European format (15/03/2026)
- **YYYY-MM-DD** - ISO format (2026-03-15)

### Week start

Choose when your calendar weeks begin:

- **Sunday** - Standard in US
- **Monday** - Standard in Europe

This affects calendar views and weekly reports.

## Language

Select your preferred display language:

- **English** - Default
- **Français** - French

Changing the language refreshes the page to apply the new setting.

## Auto-save

Profile changes save automatically as you make them. You'll see a confirmation message when changes are saved successfully.

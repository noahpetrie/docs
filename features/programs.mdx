Programs help you group related change initiatives together, enabling coordinated planning and consolidated reporting across multiple projects.

## What is a program?

A program is a collection of related initiatives that share:

- Common strategic objectives
- Overlapping stakeholder groups
- Interdependencies or sequencing requirements
- Shared resources or timelines

**Example**: A "Digital transformation" program might include:
- New CRM implementation
- Legacy system retirement
- Process automation initiative
- Digital skills training

## Program properties

| Field | Description |
|-------|-------------|
| **Name** | Clear, descriptive program title |
| **Description** | Overview of objectives and scope |
| **Status** | Active, On Hold, Completed, Archived |
| **Start date** | When the program begins |
| **Target end date** | Expected completion |

### Program statuses

- **Active** - Currently in progress
- **On Hold** - Temporarily paused
- **Completed** - Successfully finished
- **Archived** - Historical record

## Creating a program

1. Navigate to **Programs** in the sidebar
2. Click **New Program**
3. Enter program details:
   - **Name**: Clear, descriptive title
   - **Description**: Overview of program objectives and scope
   - **Start date**: When activities begin
   - **Target end date**: Expected completion
4. Click **Create**

## Managing programs

### Adding initiatives to a program

When creating or editing an initiative:

1. Open the initiative settings
2. In the **Program** field, select the parent program
3. Save the changes

### Viewing program contents

1. Open a program from the list
2. See all linked initiatives
3. View aggregate metrics:
   - Total initiatives
   - Overall status
   - Combined stakeholder count
   - Impact summary

## Program-level insights

Programs provide consolidated views across their initiatives:

### Stakeholder overlap

Identify stakeholders affected by multiple initiatives within the program. This helps:

- Prevent communication fatigue
- Coordinate messaging timing
- Understand cumulative change burden

### Impact aggregation

See the combined change impact across all program initiatives:

- Total impacts by severity
- Impacts by category
- Cross-initiative dependencies

### Timeline coordination

Visualize initiative timelines together to:

- Identify scheduling conflicts
- Optimize sequencing
- Balance resource demands

## Program administration

### Editing programs

1. Open the program
2. Click the edit icon
3. Update name, description, dates, or status
4. Save changes

### Archiving programs

Archive completed programs while preserving data:

1. Open the program
2. Click **Archive**
3. Confirm the action

Archived programs can be restored from the archived view.

### Deleting programs

<Warning>
  Deleting a program does not delete its initiatives. Initiatives will become unlinked from any program.
</Warning>

1. Open the program
2. Click delete
3. Confirm the deletion

## Access control

Programs follow the same access model as initiatives:

- **Organization members** can view programs
- **Admins** can create, edit, and delete programs

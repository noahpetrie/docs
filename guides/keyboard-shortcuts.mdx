Leap Change includes comprehensive keyboard shortcuts and a command palette to help you work faster.

## Command palette

The command palette provides quick access to navigation, actions, and search across your entire workspace.

### Opening the command palette

Press `⌘K` (Mac) or `Ctrl+K` (Windows/Linux) from anywhere in the application.

### Features

- **Navigation** - Jump to any page instantly
- **Quick actions** - Create stakeholders, impacts, surveys, tasks
- **Search** - Find stakeholders, projects, and change impacts by name
- **Theme toggle** - Switch between light and dark mode
- **Sidebar control** - Toggle sidebar visibility

### Using the command palette

1. Press `⌘K` / `Ctrl+K` to open
2. Start typing to filter commands and records
3. Use arrow keys to navigate results
4. Press Enter to execute the selected action
5. Press Escape to close

## Navigation shortcuts

Jump directly to pages using `G` followed by a second key:

| Shortcut | Destination |
|----------|-------------|
| `G` then `H` | Home dashboard |
| `G` then `P` | Projects |
| `G` then `S` | Stakeholders |
| `G` then `C` | Change Impacts |
| `G` then `V` | Surveys |
| `G` then `M` | Communications |
| `G` then `K` | Kanban board |
| `G` then `R` | Reports |
| `G` then `T` | Settings |
| `G` then `B` | Billing settings |
| `G` then `I` | Inbox |

## Action shortcuts

Create new items using `C` followed by a second key:

| Shortcut | Action |
|----------|--------|
| `C` then `T` | Create task |
| `C` then `C` | Create change impact |
| `C` then `S` | Create stakeholder |
| `C` then `P` | Create project |
| `C` then `V` | Create survey |

## Tips for efficiency

<AccordionGroup>
  <Accordion title="Use the command palette for search">
    The command palette searches across stakeholders, projects, programs, and change impacts. Type any part of the name to filter results.
  </Accordion>
  <Accordion title="Chain navigation shortcuts">
    Navigation shortcuts use a two-key sequence. Press `G`, release, then press the second key within one second.
  </Accordion>
  <Accordion title="Context-aware N key">
    The `N` key creates the appropriate item based on your current page - tasks on Kanban, projects on Projects, etc.
  </Accordion>
  <Accordion title="Quick theme switching">
    Press `T` from anywhere to toggle between light and dark mode without opening the command palette.
  </Accordion>
</AccordionGroup>

## Viewing all shortcuts

Press `⌘?` (Mac) or `Ctrl+?` (Windows/Linux) to open the keyboard shortcuts reference modal at any time.

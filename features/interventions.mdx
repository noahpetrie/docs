Leap Change includes a library of proven change management interventions to help you plan effective stakeholder engagement strategies.

## What are interventions?

Interventions are structured activities designed to support people through change. They help:

- Build awareness of the change
- Develop necessary skills and knowledge
- Address concerns and resistance
- Reinforce new behaviors
- Sustain change over time

## Intervention categories

### Communication

Information-sharing activities:

| Intervention | Description | Best for |
|--------------|-------------|----------|
| **Town hall** | All-hands meeting with leadership | Major announcements |
| **Newsletter** | Regular written updates | Ongoing awareness |
| **FAQ document** | Answers to common questions | Reducing uncertainty |
| **Video message** | Pre-recorded leadership update | Consistent messaging |
| **Intranet update** | Posted information on company portal | Reference materials |

### Training

Skill and knowledge development:

| Intervention | Description | Best for |
|--------------|-------------|----------|
| **Classroom training** | Instructor-led session | Complex skills |
| **E-learning** | Self-paced online course | Scalable delivery |
| **Job aids** | Quick reference guides | On-the-job support |
| **Hands-on workshop** | Practice-based learning | Technical skills |
| **Lunch and learn** | Informal knowledge sharing | Low-impact changes |

### Engagement

Two-way dialogue and involvement:

| Intervention | Description | Best for |
|--------------|-------------|----------|
| **Focus group** | Small group discussion | Understanding concerns |
| **One-on-one meeting** | Individual conversation | High-influence stakeholders |
| **Feedback session** | Structured input gathering | Process improvement |
| **Change champion network** | Peer advocates | Grassroots support |
| **Executive sponsor meeting** | Leadership alignment | Senior stakeholders |

### Support

Ongoing assistance and reinforcement:

| Intervention | Description | Best for |
|--------------|-------------|----------|
| **Help desk** | Dedicated support channel | Technical issues |
| **Coaching** | Individualized guidance | Behavioral change |
| **Peer mentoring** | Colleague-to-colleague support | Knowledge transfer |
| **Drop-in sessions** | Open Q&A availability | Accessible support |
| **Success celebration** | Recognition of progress | Reinforcement |

## Using interventions in Leap Change

### Communication planning

When creating communication activities:

1. Consider the intervention type needed
2. Match to stakeholder preferences
3. Schedule appropriately in your plan
4. Track completion and effectiveness

### Impact mitigation

For each change impact:

1. Review the affected stakeholders
2. Identify appropriate interventions
3. Add mitigation actions using intervention types
4. Assign owners and due dates

### Stakeholder-specific strategies

Consider stakeholder characteristics:

- **High influence, low support** - Executive meetings, one-on-ones
- **High influence, high support** - Champion network, sponsor meetings
- **Low influence, low support** - Focus groups, FAQ documents
- **Low influence, high support** - Training, success celebrations

## Selecting interventions

### By change phase

| Phase | Focus | Interventions |
|-------|-------|---------------|
| **Awareness** | Understanding the change | Town halls, newsletters, FAQs |
| **Desire** | Building motivation | One-on-ones, sponsor meetings |
| **Knowledge** | Learning new skills | Training, e-learning, workshops |
| **Ability** | Applying skills | Job aids, coaching, help desk |
| **Reinforcement** | Sustaining change | Celebrations, feedback sessions |

### By impact severity

| Severity | Intervention intensity |
|----------|------------------------|
| **High** | Multiple touchpoints, personalized engagement |
| **Medium** | Group sessions, targeted communications |
| **Low** | General communications, self-service resources |

The Home dashboard is your starting point in Leap Change, providing an overview of your change management activities and quick access to key features.

## Dashboard overview

When you sign in, you'll see:

- **Personalized greeting** - Based on time of day and your name
- **Statistics cards** - Quick counts of stakeholders, impacts, surveys, and AI tools
- **Quick action cards** - One-click access to common tasks
- **Onboarding progress** - New user guidance (if applicable)

## Statistics cards

Four cards show your current counts:

| Card | What it shows |
|------|---------------|
| **Stakeholders** | Total stakeholders across all projects |
| **Change Impacts** | Active impacts being tracked |
| **Surveys** | Surveys created |
| **AI Tools** | Available AI features |

Click any card to navigate to that section.

## Quick actions

The dashboard provides shortcuts to:

- **Add Stakeholder** - Create a new stakeholder
- **Create Impact** - Log a change impact
- **Launch Survey** - Start a readiness assessment
- **View Reports** - Access analytics
- **AI Features** - Explore AI capabilities

## Onboarding progress

New users see an onboarding tracker with steps to get started:

1. **Add your first stakeholder** - Import or create a stakeholder
2. **Record your first change impact** - Document what's changing
3. **Explore AI features** - Try transcript extraction or chat
4. **Generate your first report** - Create an analytics view

The progress tracker disappears once all steps are completed.

## Navigating from home

Use the sidebar to navigate to:

- **Projects** - Manage change initiatives
- **Stakeholders** - View and manage stakeholders
- **Change Impacts** - Track identified impacts
- **Surveys** - Create and manage surveys
- **Communications** - Plan stakeholder communications
- **Tasks** - Kanban board for work items
- **Reports** - Analytics and visualizations
- **AI Features** - AI-powered tools
- **Settings** - Account and organization configuration

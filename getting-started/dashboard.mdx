The dashboard is your starting point in Leap Change, providing an overview of your change initiatives and quick access to common actions.

## Dashboard sections

### Welcome banner

A personalized greeting based on the time of day, along with your organization name and current date.

### Key metrics

At-a-glance statistics for your change initiatives:

| Metric | Description |
|--------|-------------|
| **Stakeholders** | Total stakeholders across all projects |
| **Change impacts** | Number of identified change impacts |
| **Active surveys** | Surveys currently accepting responses |
| **Open tasks** | Tasks in progress or pending |

Click any metric to navigate directly to that section.

### Onboarding checklist

For new users, the dashboard displays an onboarding checklist to help you get started:

1. **Add your first stakeholder** - Create or import stakeholders
2. **Record a change impact** - Document your first change impact
3. **Explore AI assistant** - Try the AI-powered features
4. **Generate a report** - Create your first report

Complete these steps to familiarize yourself with the platform. The checklist disappears once all items are completed.

### Quick actions

Common tasks accessible from the dashboard:

- **New project** - Create a change initiative
- **Add stakeholder** - Add a new stakeholder
- **Create survey** - Launch a readiness survey
- **View reports** - Access analytics

### Recent activity

See recent updates across your organization:

- Newly created stakeholders
- Updated change impacts
- Survey responses received
- Task completions

## Customizing your view

### Project filter

Focus the dashboard on specific projects:

1. Click the project filter dropdown
2. Select one or more projects
3. Metrics and activity update to reflect the selection

### Date range

Adjust the time period for activity and trends:

1. Click the date range selector
2. Choose a preset (Today, This week, This month)
3. Or select a custom date range

## Navigation

From the dashboard, navigate to any section using the sidebar:

- **Home** - Return to dashboard
- **Programs** - View and manage programs
- **Projects** - Manage change initiatives
- **Stakeholders** - Access stakeholder management
- **Change impacts** - View and create impacts
- **Surveys** - Manage feedback surveys
- **Communications** - Plan communications
- **Tasks** - Kanban task board
- **Reports** - Analytics and visualizations
- **AI Features** - AI assistant and tools
- **Settings** - Account and organization settings

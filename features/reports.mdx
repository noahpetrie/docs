Leap Change provides built-in reports to help you understand your change landscape, track progress, and communicate with leadership.

## Available reports

### Stakeholder map

Visualize stakeholder locations geographically:

- Plot stakeholders on an interactive map
- Filter by various attributes
- Identify geographic concentrations
- Plan location-based engagement

<Note>
  Stakeholder mapping requires location data to be populated in stakeholder profiles.
</Note>

### Stakeholder impact report

Analyze the change burden on your stakeholders:

- See which stakeholders are most affected
- Understand impact distribution
- Identify individuals with high cumulative impact
- Prioritize support interventions

### Impact report

Visualize your change impacts:

- Distribution by category
- Distribution by severity
- Status breakdown
- Trend over time

### Communication preferences

Summarize how stakeholders prefer to be reached:

- Preferred channels
- Best times
- Language preferences

## Accessing reports

1. Click **Reports** in the sidebar
2. Select the report type
3. Apply any relevant filters
4. View the visualization

## Filtering and customization

Most reports support filtering by:

- Initiative/project
- Stakeholder group
- Time period
- Status
- Severity
- Category

## Exporting data

Export report data for use in presentations or external tools:

1. Generate the report
2. Click **Export**
3. Choose format (CSV, Excel, PDF where available)
4. Download the file

## Dashboard overview

The **Dashboard** provides at-a-glance metrics:

- Active initiatives
- Total stakeholders
- Open change impacts by severity
- Upcoming communications
- Recent activity

## Organization dashboard

For organization-wide visibility:

1. Go to **Dashboard** > **Org**
2. See metrics across all initiatives
3. Compare initiative progress
4. Identify organization-level trends

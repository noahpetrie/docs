Projects are the central organizing unit for your change management work. Each project represents a distinct change initiative and contains all related stakeholders, impacts, surveys, communications, and tasks.

## What is a project?

A project in Leap Change represents a specific organizational change:

- System implementations (ERP, CRM, etc.)
- Process changes
- Organizational restructuring
- Policy updates
- Cultural transformation initiatives

## Creating a project

Leap Change guides you through project creation with a step-by-step wizard.

### Step 1: Describe the change

- **Project name** - Clear, descriptive title
- **Description** - What's changing and why
- **Change owner** - Person responsible for the change
- **Sponsor** - Executive sponsor

### Step 2: Classify

| Field | Description |
|-------|-------------|
| **Change type** | Technology, Process, Organizational, Policy |
| **Priority** | Critical, High, Medium, Low |
| **Departments** | Affected business areas |
| **Geographic scope** | Local, Regional, Global |

### Step 3: Set timeline

- **Start date** - When change activities begin
- **Target go-live** - Expected implementation date
- **Milestones** - Key checkpoints

### Step 4: Launch

Review and create your project.

## Project dashboard

Each project has a dedicated dashboard showing:

- **Overview** - Key metrics and status
- **Stakeholders** - People linked to this project
- **Change impacts** - Impacts specific to this project
- **Tasks** - Work items for this initiative
- **Communications** - Planned and completed communications

## Project team

Add team members to collaborate on the project:

1. Open the project
2. Go to **Team** section
3. Click **Add Member**
4. Select from organization members
5. Assign a role (Owner, Contributor, Viewer)

## Project lifecycle

Projects move through these stages:

| Stage | Description |
|-------|-------------|
| **Planning** | Initial setup and assessment |
| **Active** | Change activities underway |
| **Closing** | Wrapping up and sustainment |
| **Archived** | Completed, read-only |

## Linking to programs

Connect projects to programs for portfolio-level visibility:

1. Edit the project
2. Select a **Program** from the dropdown
3. Save changes

Projects in a program share:
- Consolidated reporting
- Stakeholder overlap analysis
- Timeline coordination

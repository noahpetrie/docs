Project settings let you update project details, manage team access, and configure project-specific options.

## Accessing settings

1. Open the project you want to configure
2. Click the **Settings** icon or go to the settings tab
3. Make changes and save

## Basic information

Update core project details:

- **Project name** - Clear, descriptive title
- **Description** - What the change is about and why
- **Change owner** - Primary person responsible
- **Sponsor** - Executive sponsor

## Timeline settings

Manage project dates:

- **Start date** - When work begins
- **Target go-live** - Expected completion
- **Status** - Current project status

### Changing status

| From | To | When to use |
|------|----|-------------|
| Active | On Hold | Temporarily pausing the initiative |
| Active | Completed | Successfully finished |
| On Hold | Active | Resuming paused work |
| Completed | Archived | Moving to historical record |

## Project metadata

Configure additional project attributes:

### Classification

- **Change type** - Technology, Process, Organizational, Policy
- **Priority** - Critical, High, Medium, Low
- **Departments affected** - Select relevant business units
- **Geographic scope** - Local, Regional, Global

### Unique identifier

Each project has an auto-generated unique ID (e.g., PRJ-2026-00001) for reference and tracking.

## Program assignment

Link the project to a program:

1. In settings, find the **Program** field
2. Select an existing program or create a new one
3. Save changes

Projects can belong to one program at a time. Remove program assignment by selecting "None."

## Archiving projects

Archive completed projects to keep data while removing them from active views:

1. Open project settings
2. Click **Archive Project**
3. Confirm the action

<Note>
  Archived projects are not deleted. All data is preserved and can be accessed from the archived projects view.
</Note>

### Restoring archived projects

1. Go to **Projects** → **Archived**
2. Find the project
3. Click **Restore**
4. Project returns to active status

## Deleting projects

<Warning>
  Deleting a project permanently removes all associated data including stakeholder assignments, impacts, surveys, and communications. This action cannot be undone.
</Warning>

To delete a project:

1. Open project settings
2. Scroll to **Danger Zone**
3. Click **Delete Project**
4. Type the project name to confirm
5. Click **Delete Forever**

The Microsoft Teams integration enables direct communication with stakeholders and monitoring of Teams channels for change-related discussions.

## Features

### Direct messaging

Send Teams messages to stakeholders:

- Compose messages from stakeholder profiles
- Use as part of communication activities
- Send survey invitations via Teams
- Track message delivery

### Channel monitoring

Monitor Teams channels for change-related discussions:

- Configure keywords to watch
- Receive notifications when mentioned
- Capture insights for impact analysis
- Track stakeholder sentiment

### Teams bot

The Leap Change bot can:

- Respond to queries about changes
- Provide stakeholder lookup
- Send scheduled notifications
- Support interactive workflows

## Prerequisites

- Microsoft Teams enabled for your organization
- Microsoft 365 integration connected (recommended)
- Leap Change admin role

## Connecting Teams

1. Go to **Settings** → **Integrations**
2. Find **Microsoft Teams** and click **Connect**
3. Sign in with your Microsoft account
4. Authorize the Teams bot
5. Select default channels (optional)

### Admin consent

Some features require admin consent:

- Sending messages on behalf of users
- Accessing channel content for monitoring
- Bot installation across the organization

Contact your IT admin if you cannot enable these features.

## Sending messages

### From a stakeholder profile

1. Open a stakeholder's profile
2. Click **Send Teams Message**
3. Compose your message
4. Click **Send**

### As a communication activity

1. Create or edit a communication activity
2. Set type to **Teams Message**
3. Select target stakeholders
4. Compose the message
5. Schedule or send immediately

### Bulk messaging

1. Select multiple stakeholders
2. Click **Bulk Actions** → **Send Teams Message**
3. Compose a single message
4. Send to all selected stakeholders

## Channel monitoring

### Setting up monitoring

1. Go to **Settings** → **Integrations** → **Microsoft Teams**
2. Click **Configure Channels**
3. Select channels to monitor
4. Add keywords to watch for
5. Save configuration

### Keywords

Add keywords related to your changes:

- Project names
- System names
- Change terminology
- Leadership names

Leap Change notifies you when these keywords appear.

### Viewing mentions

1. Go to **Home** or the relevant project
2. Check the **Channel Mentions** section
3. Review captured discussions
4. Link relevant insights to impacts

## Installing the Teams app

For full bot functionality:

1. Download the Leap Change Teams app package
2. Upload to your Teams admin center
3. Deploy to users or allow self-installation
4. Users can interact with the bot in Teams

## Troubleshooting

### Messages not sending

- Verify the stakeholder has a valid Teams account
- Check that the email matches their Teams identity
- Ensure the integration is properly connected

### Channel monitoring not working

- Confirm admin consent was granted
- Verify channel access permissions
- Check that keywords are configured

### Bot not responding

- Ensure the app is installed in Teams
- Check that the user has interacted with the bot
- Verify the integration connection is active

Personalize how Leap Change looks and feels with theme and color options.

## Accessing appearance settings

1. Click the **Settings** icon in the sidebar
2. Select **Appearance** from the menu

## Theme selection

Choose your preferred color scheme:

| Theme | Description |
|-------|-------------|
| **Light** | White backgrounds with dark text |
| **Dark** | Dark backgrounds with light text |
| **System** | Automatically matches your device settings |

Click a theme option to apply it immediately.

## Accent color

Select a primary accent color used throughout the interface:

| Color | Use case |
|-------|----------|
| Blue | Default professional look |
| Purple | Creative or modern feel |
| Green | Natural or growth-focused |
| Orange | Energetic or action-oriented |
| Red | Bold or urgent emphasis |
| Yellow | Warm or optimistic tone |

The accent color appears on:

- Primary buttons
- Active navigation items
- Progress indicators
- Interactive elements
- Links and highlights

## Preview

Theme and color changes apply instantly, letting you preview the look before leaving the settings page.

<Tip>
  Dark mode can reduce eye strain when working in low-light environments or for extended periods.
</Tip>

The Kanban board provides a visual way to manage tasks across your change initiatives.

## Board layout

The board displays tasks in columns by status:

| Column | Purpose |
|--------|----------|
| **Planned** | Tasks identified but not started |
| **Up next** | Prioritized for upcoming work |
| **In progress** | Tasks currently being worked on |
| **In review** | Completed, awaiting approval |
| **Done** | Successfully completed |
| **Canceled** | Tasks no longer needed |

## Navigating the board

1. Go to **Tasks** in the sidebar
2. The Kanban view is the default
3. Scroll horizontally to see all columns
4. Scroll vertically within columns for more tasks

## Moving tasks

### Drag and drop

1. Click and hold a task card
2. Drag to the target column
3. Drop in the desired position
4. The status updates automatically

### Quick actions

1. Click the task card
2. Use the status dropdown
3. Select the new status
4. Close the task details

## Task cards

Each card displays:

- **Title** - Task name
- **Assignee** - Avatar of the assigned person
- **Project** - Associated initiative (if linked)
- **Priority** - Visual indicator
- **Due date** - Target completion (with overdue highlighting)
- **Labels** - Category tags

## Filtering tasks

Filter the board to focus on specific tasks:

### By assignee
- Click **Filter** → **Assignee**
- Select team members
- View only their tasks

### By project
- Click **Filter** → **Project**
- Select initiatives
- See tasks for those projects

### By label
- Click **Filter** → **Labels**
- Select one or more labels
- View matching tasks

### By priority
- Click **Filter** → **Priority**
- Select priority levels
- Focus on high-priority work

## Creating tasks

Add tasks directly from the board:

1. Click **+** in any column header
2. Enter the task title
3. Press Enter to create
4. Click the card to add details

## Reordering tasks

Change task order within a column:

1. Drag the task up or down
2. Drop at the desired position
3. Order is saved automatically

Task order is preserved per column and persists across sessions.

## Keyboard shortcuts

| Shortcut | Action |
|----------|--------|
| `N` | Create new task |
| `Enter` | Open selected task |
| `Esc` | Close task details |

## Best practices

<Tip>
  Keep "In progress" column small. Too many concurrent tasks indicates overcommitment.
</Tip>

### Work in progress limits

- Limit concurrent tasks per person
- Focus on finishing over starting
- Move tasks to Done promptly

### Regular grooming

- Review Planned column weekly
- Prioritize Up next based on goals
- Archive completed tasks periodically

### Visual management

- Use labels consistently
- Keep titles scannable
- Update assignees promptly

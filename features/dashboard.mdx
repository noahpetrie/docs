The Dashboard is your central hub in Leap Change. It provides an at-a-glance view of your change management activities and quick access to common tasks.

## Home page

When you sign in, you'll land on the Home page which shows:

### Welcome section

- Personalized greeting
- Quick action buttons
- Getting started guidance for new users

### Key metrics

| Metric | Description |
|--------|-------------|
| **Total stakeholders** | Number of stakeholders across all projects |
| **Total impacts** | Change impacts documented |
| **Total surveys** | Surveys created |
| **Onboarding status** | Whether you've completed initial setup |

### Quick actions

- Add stakeholder
- Create change impact
- Send survey
- Create communication plan

## Organization dashboard

Access the organization dashboard from the sidebar to view all projects.

### Active projects

Displays your current change initiatives as cards showing:

- Project name and description
- Status badge (Active, On Hold, Completed)
- Stakeholder count
- Impact count
- Progress indicators

### Project card actions

- **Click** to open project details
- **Menu (...)** to access quick actions:
  - Edit project
  - Archive project
  - Duplicate project

### Archived projects

Toggle to view archived projects:

- Previously completed initiatives
- Cancelled projects
- Historical data preserved
- Can be restored if needed

## Filtering and sorting

### Filter projects

- **By status** - Active, On Hold, Completed
- **By priority** - Critical, High, Medium, Low
- **Search** - Find by name or description

### Sort options

- Recently updated
- Alphabetical
- Start date
- End date
- Priority

## Creating new content

From the dashboard, you can quickly create:

### New project

Click **New Project** to launch the project creation wizard.

### New program

Navigate to Programs and click **New Program** to group related projects.

## Navigation

From the dashboard, navigate to:

| Section | Purpose |
|---------|----------|
| **Projects** | View and manage individual projects |
| **Programs** | Manage project portfolios |
| **Stakeholders** | View all stakeholders |
| **Change Impacts** | View all impacts |
| **Surveys** | Manage readiness surveys |
| **Communications** | Plan stakeholder communications |
| **Tasks** | Kanban board for work items |
| **Reports** | Analytics and visualizations |
| **Settings** | Account and organization settings |

## Onboarding

New users see onboarding guidance:

1. **Create your first project** - Set up a change initiative
2. **Add stakeholders** - Import or manually add people
3. **Document impacts** - Identify what's changing
4. **Assess readiness** - Create and send surveys

Once you have stakeholders and impacts, the onboarding guidance is replaced with your activity dashboard.

The Home dashboard provides a personalized overview of your change management activities with quick access to common actions and key metrics.

## Accessing the dashboard

Click **Home** in the sidebar or use the keyboard shortcut `G` then `H` to navigate to your dashboard.

## Dashboard components

### Personalized greeting

The dashboard displays a time-based greeting with your name:

- Good morning (before 12 PM)
- Good afternoon (12 PM - 5 PM)
- Good evening (after 5 PM)

### Quick stats

View key metrics at a glance:

| Metric | Description |
|--------|-------------|
| **Stakeholders** | Total number of stakeholders in your organization |
| **Change Impacts** | Total change impacts across all projects |
| **Surveys** | Number of surveys created |
| **AI Tools** | Available AI-powered features |

### Quick actions

Access common tasks directly from the dashboard:

- Create a new project
- Add a stakeholder
- Create a change impact
- Start a survey

## Getting started checklist

New users see an onboarding checklist to help them get started:

1. **Create your organization** - Set up your workspace
2. **Add stakeholders** - Import or create your first stakeholders
3. **Create a project** - Launch your first change initiative
4. **Document impacts** - Identify change impacts
5. **Send a survey** - Assess stakeholder readiness

The checklist disappears once you've added stakeholders and change impacts.

## Keyboard navigation

From the Home dashboard, use these shortcuts:

| Shortcut | Action |
|----------|--------|
| `⌘K` / `Ctrl+K` | Open command palette |
| `G` then `P` | Go to Projects |
| `G` then `S` | Go to Stakeholders |
| `G` then `C` | Go to Change Impacts |
| `G` then `V` | Go to Surveys |

## Customization

The dashboard adapts based on your activity:

- Shows relevant quick actions based on your workflow
- Displays progress indicators for ongoing initiatives
- Highlights items requiring attention

Customize Leap Change's appearance to match your preferences with theme options and accent colors.

## Accessing appearance settings

1. Click your avatar in the top navigation
2. Select **Settings**
3. Click **Appearance** in the sidebar

## Accent color

Choose a custom accent color for buttons, links, and highlights:

1. Find the **Accent Color** section
2. Click the color picker or enter a hex code
3. Preview the color in the interface
4. Click **Save** to apply

### Color suggestions

| Color | Hex code | Best for |
|-------|----------|----------|
| Default green | #0C8C5E | Standard Leap Change theme |
| Blue | #2563EB | Corporate, professional |
| Purple | #7C3AED | Creative teams |
| Orange | #EA580C | Energetic, attention-grabbing |

### Reset to default

Click **Reset** to restore the original Leap Change accent color.

## Display preferences

### Compact mode

Enable compact mode for denser information display:

- Reduces spacing between elements
- Shows more content on screen
- Ideal for experienced users with large monitors

### Sidebar behavior

Control sidebar display:

| Option | Behavior |
|--------|----------|
| **Expanded** | Sidebar always visible |
| **Collapsed** | Sidebar minimized, hover to expand |
| **Auto** | Collapses on small screens |

## Saving preferences

Appearance settings sync across all your devices when you sign in.

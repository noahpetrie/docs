---
title: Managing initiatives
description: Add and organize initiatives within programs
---

Once you've created a program, you can link change initiatives to group them together.

## Adding initiatives to a program

When creating a new initiative:

1. Navigate to **Projects** → **New Project**
2. Complete the project wizard
3. In the **Program** field, select the parent program
4. Finish creating the project

For existing initiatives:

1. Open the initiative settings
2. Find the **Program** field
3. Select the parent program
4. Save the changes

## Viewing program contents

1. Navigate to **Programs**
2. Click on a program name
3. View all linked initiatives in a list
4. See aggregate metrics:
   - Total initiatives count
   - Overall program status
   - Combined stakeholder count
   - Impact summary across all initiatives

## Managing initiative order

Within a program view, you can:

- Sort initiatives by name, status, or dates
- Filter by initiative stage
- Search for specific initiatives

## Removing initiatives from a program

1. Open the initiative
2. Navigate to settings
3. Clear the **Program** field
4. Save changes

The initiative remains in your organization but is no longer linked to the program.

## Program reporting

Programs provide consolidated views:

### Stakeholder analysis

- See stakeholders affected by multiple initiatives
- Identify change saturation risks
- Plan coordinated engagement

### Impact overview

- Combined impact counts by severity
- Cross-initiative dependencies
- Aggregate mitigation status

### Timeline view

- Overlapping initiative timelines
- Milestone alignment
- Resource conflicts

## Best practices

<Tip>
  Review program contents regularly to ensure all initiatives are properly linked and the program scope remains appropriate.
</Tip>

### Keep programs focused

- Limit to 5-10 related initiatives
- Ensure clear thematic connection
- Split large programs if needed

### Monitor dependencies

- Track cross-initiative dependencies
- Sequence initiatives appropriately
- Update timelines when dependencies shift

### Coordinate stakeholders

- Identify overlapping audiences
- Stagger high-impact communications
- Share resources where appropriate

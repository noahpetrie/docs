The Home dashboard provides an at-a-glance view of your change management activities and key metrics.

## Dashboard overview

When you sign in, the Home dashboard displays:

- **Key metrics** - Quick stats on stakeholders, impacts, and surveys
- **Recent activity** - Latest changes and updates
- **Getting started** - Onboarding checklist for new users
- **Quick actions** - Common tasks you can start immediately

## Key metrics

The dashboard shows real-time counts:

| Metric | Description |
|--------|-------------|
| **Stakeholders** | Total stakeholders in your organization |
| **Change impacts** | Active impacts across all projects |
| **Open surveys** | Surveys currently accepting responses |
| **Active projects** | Projects currently in progress |

Click any metric to navigate to the detailed view.

## Getting started checklist

New users see an onboarding checklist:

1. **Create your first project** - Set up a change initiative
2. **Add stakeholders** - Import or manually add people
3. **Document impacts** - Identify what's changing
4. **Send a survey** - Assess stakeholder readiness
5. **Plan communications** - Schedule stakeholder touchpoints

Complete each step to build your change management foundation.

## Quick actions

Start common tasks directly from the dashboard:

- **New project** - Create a change initiative
- **Add stakeholder** - Add a new person
- **Create impact** - Document a change impact
- **Launch survey** - Start collecting feedback

## Recent activity

See what's happening across your organization:

- New impacts added
- Surveys launched or completed
- Stakeholder updates
- Communication activities

Activity is filtered to show only items relevant to your projects.

## Customizing your dashboard

The dashboard adapts based on:

- **Your role** - Admins see organization-wide stats
- **Your projects** - Members see project-specific data
- **Recent focus** - Frequently accessed items appear prominently

## Navigating from the dashboard

Use the sidebar to access:

| Section | Purpose |
|---------|----------|
| **Home** | Return to the dashboard |
| **Projects** | Manage change initiatives |
| **Stakeholders** | View and manage stakeholders |
| **Change Impacts** | Track identified impacts |
| **Surveys** | Create and manage surveys |
| **Communications** | Plan stakeholder communications |
| **Tasks** | Kanban board for work items |
| **Reports** | Analytics and visualization |
| **Settings** | Account and organization settings |

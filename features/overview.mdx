## Platform capabilities

| Capability | Description |
|------------|-------------|
| **Multi-tenant** | Secure, isolated workspaces for your organization |
| **Role-based access** | Admin and member roles with appropriate permissions |
| **Integrations** | Connect with Microsoft 365, Teams, and Slack |
| **Import/Export** | Bulk data operations for stakeholders and reports |
| **Localization** | English and French language support |

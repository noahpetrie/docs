---
title: Managing initiatives
description: Link projects to programs and manage program membership
---

Connect your change initiatives to programs for coordinated management and consolidated reporting.

## Adding initiatives to a program

### When creating an initiative

1. Start the new project wizard
2. In the project details, select the **Program** field
3. Choose the parent program
4. Complete project creation

### For existing initiatives

1. Open the project settings
2. Find the **Program** field
3. Select the target program
4. Save changes

## Viewing program initiatives

1. Navigate to **Programs**
2. Click on a program
3. View all linked initiatives with:
   - Project name and status
   - Timeline
   - Key metrics
   - Quick actions

## Removing initiatives from a program

1. Open the project settings
2. Clear the **Program** field or select "None"
3. Save changes

The initiative remains but is no longer associated with the program.

## Cross-initiative coordination

### Stakeholder overlap

Programs automatically identify stakeholders affected by multiple initiatives:

1. Open the program
2. View the **Stakeholder Overlap** section
3. See which stakeholders appear in multiple projects
4. Use this to:
   - Coordinate communications
   - Prevent change fatigue
   - Identify high-impact individuals

### Timeline coordination

See all initiative timelines together:

- Identify scheduling conflicts
- Find busy periods for stakeholders
- Plan sequencing strategically
- Balance resource demands

### Impact aggregation

View combined change impacts across the program:

- Total impacts by severity
- Category distribution
- Status across initiatives
- Dependencies between impacts

## Program-level actions

From the program view, you can:

- **Export program report** - Consolidated PDF or CSV
- **View combined dashboard** - Aggregate metrics
- **Navigate to initiatives** - Quick access to any project
- **Edit program details** - Update name and description

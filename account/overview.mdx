Access account settings to manage your profile, customize preferences, and configure your organization.

## Accessing settings

1. Click your profile avatar in the top navigation
2. Select **Settings** from the dropdown
3. Navigate between setting categories using the sidebar

## Quick actions

From any settings page, you can:

- **Save changes** - Apply your modifications
- **Reset** - Revert to previous values
- **Cancel** - Exit without saving

## Role-based access

Some settings require specific roles:

| Setting area | Required role |
|--------------|---------------|
| Profile | Any user |
| Appearance | Any user |
| Notifications | Any user |
| Organization | Admin |
| Team management | Admin |
| Billing | Admin |
| Integrations | Admin |

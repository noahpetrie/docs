Configure your personal profile and preferences in Leap Change.

## Accessing profile settings

1. Click **Settings** in the sidebar
2. Select **Profile** from the settings menu

## Personal information

### Basic details

Update your account information:

| Field | Description |
|-------|-------------|
| **First name** | Your first name |
| **Last name** | Your last name |
| **Email** | Your login email (read-only) |

### Profile picture

Upload an avatar image:

1. Click **Upload image**
2. Select an image file (PNG, JPEG, or GIF)
3. Maximum file size: 10MB
4. Recommended: Square images for best display

### Additional details

| Field | Description |
|-------|-------------|
| **Job role** | Your position or title |
| **Company** | Your organization name |
| **Phone** | Contact phone number |
| **Country** | Your location |
| **LinkedIn** | Your LinkedIn profile URL |

## Time preferences

Configure how dates and times display throughout Leap Change:

### Timezone

Set your local timezone for accurate scheduling and notifications. The platform uses your browser's timezone by default.

### Time format

Choose between:

- **12-hour** (e.g., 2:30 PM)
- **24-hour** (e.g., 14:30)

### Date format

Select your preferred date display:

- **MM/DD/YYYY** (US format)
- **DD/MM/YYYY** (International format)
- **YYYY-MM-DD** (ISO format)

### Week start

Set your calendar week start day:

- **Sunday** (US convention)
- **Monday** (International convention)

## Language

Change the display language for the entire platform:

- **English**
- **Français** (French)

<Note>
  Changing the language reloads the page to apply the new setting.
</Note>

## Auto-save

All profile changes are saved automatically as you make them. There's no need to click a save button.

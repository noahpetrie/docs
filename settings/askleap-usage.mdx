Track your organization's AI feature usage in Leap Change.

## Accessing usage settings

1. Click **Settings** in the sidebar
2. Select **AskLeap Usage** from the settings menu

## Usage overview

View your current AI usage:

### Current period

- **Requests used** - Number of AI requests made this billing cycle
- **Requests remaining** - Available requests until the quota resets
- **Usage percentage** - Visual progress bar of consumption

### Usage breakdown

See how AI requests are distributed across features:

| Feature | Description |
|---------|-------------|
| **Chat queries** | Questions asked to the AskLeap assistant |
| **Impact extraction** | Transcripts analyzed for change impacts |
| **Impact analysis** | AI insights generated for impacts |
| **Stakeholder analysis** | AI-powered stakeholder recommendations |

## Usage history

View historical usage patterns:

- **Daily usage** - Requests per day over the past month
- **Feature distribution** - Which features use the most requests
- **Trend analysis** - Increasing or decreasing usage over time

## Plan limits

AI quotas vary by plan:

| Plan | Monthly AI requests |
|------|---------------------|
| **Basic** | 100 |
| **Pro** | 1,000 |
| **Enterprise** | Unlimited |

## Quota management

### Approaching quota

When nearing your limit:

- A warning appears in the usage dashboard
- AI features show remaining requests
- Consider upgrading for more capacity

### Quota exceeded

When the quota is reached:

- AI features become temporarily unavailable
- Existing AI-generated content remains accessible
- Quota resets on your billing cycle date

### Upgrading

1. Go to **Settings** → **Billing**
2. Click **Upgrade plan**
3. Select a plan with higher AI limits
4. New quota is available immediately

<Tip>
  Use transcript extraction for the highest time savings—it typically identifies impacts you might have missed manually.
</Tip>

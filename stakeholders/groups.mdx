Stakeholder groups help you organize stakeholders for easier management and targeted communications.

## Group types

Leap Change supports four group types:

| Type | Use case |
|------|----------|
| **Functional** | Group by job function (e.g., Managers, IT Support) |
| **Geographic** | Group by location (e.g., EMEA, North America) |
| **Hierarchical** | Group by org level (e.g., Executive, Middle Management) |
| **Custom** | Any grouping that fits your needs |

## Creating groups

1. Navigate to **Stakeholders** → **Groups**
2. Click **Create Group**
3. Enter group details:
   - **Name** - Descriptive group name
   - **Type** - Select from functional, geographic, hierarchical, or custom
   - **Description** - Optional details about the group
4. Click **Create**

## Adding members

### From the group

1. Open the group
2. Click **Add Members**
3. Search and select stakeholders
4. Click **Add**

### From stakeholder profiles

1. Open a stakeholder profile
2. Go to the **Groups** section
3. Click **Add to Group**
4. Select one or more groups

### Bulk assignment

1. Go to **Stakeholders** table view
2. Select multiple stakeholders using checkboxes
3. Click **Bulk Actions** → **Add to Group**
4. Choose the target group

## Using groups

Groups are useful for:

- **Filtering** - Filter stakeholder tables by group
- **Surveys** - Send surveys to specific groups
- **Communications** - Target messages to groups
- **Change impacts** - Link impacts to affected groups
- **Reports** - Analyze engagement by group

## Microsoft 365 group sync

With Microsoft 365 integration, Entra ID groups can sync to stakeholder groups automatically. This keeps your stakeholder groups aligned with your organizational structure.

Configure group sync in **Settings** → **Integrations** → **Microsoft 365**.

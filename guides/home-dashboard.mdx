The Home dashboard provides a quick overview of your change management activities and quick access to common actions.

## Accessing the dashboard

Navigate to **Home** in the sidebar, or press `G` then `H` from anywhere in the application.

## Dashboard overview

The dashboard displays key metrics at a glance:

### Quick stats

| Metric | Description |
|--------|-------------|
| **Total stakeholders** | Number of stakeholders in your organization |
| **Change impacts** | Number of identified change impacts |
| **Active surveys** | Surveys currently collecting responses |

## Getting started

If you're new to Leap Change, the dashboard guides you through initial setup:

### Onboarding steps

1. **Add stakeholders** - Import or create your first stakeholders
2. **Document impacts** - Identify changes affecting your organization
3. **Create surveys** - Assess stakeholder readiness

Once you've added stakeholders and impacts, the dashboard switches to showing your active metrics.

## Quick actions

The dashboard provides shortcuts to common tasks:

- Create a new project
- Add stakeholders
- Document a change impact
- Send a survey

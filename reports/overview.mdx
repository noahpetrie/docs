## Accessing reports

1. Navigate to **Reports** in the sidebar
2. Select a report from the gallery
3. Apply filters as needed
4. View visualizations and data

## Common features

### Filtering

All reports support filtering by:

- **Project** - Focus on specific change initiatives
- **Date range** - Analyze specific time periods
- **Department** - Filter by organizational unit
- **Groups** - Filter by stakeholder groups
- **Status** - Include or exclude based on status

### Export

Export report data:

- **PDF** - Formatted report with visualizations for presentations
- **CSV** - Raw data for external analysis in spreadsheets
- **Image** - Charts as PNG files for documents

### Drill-down

Click chart elements to:

- See underlying data
- Navigate to individual records
- Apply the segment as a filter

### AI analysis

Some reports include AI-powered insights:

- **Stakeholder Map** - Geographic analysis, relationship insights, risk identification
- **Impact Report** - Pattern detection and recommendations

## Report gallery

The Reports page shows all available reports with:

- **Preview** - Thumbnail of the report
- **Description** - What the report shows
- **Last viewed** - When you last accessed it
- **Quick filters** - Common filter presets

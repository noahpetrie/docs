---
title: Managing program projects
description: Add, organize, and track projects within programs
---

Link change initiatives to programs to organize related work and enable portfolio-level tracking.

## Adding projects to a program

### From the program

1. Open the program
2. Click **Add Project**
3. Select from existing projects
4. Click **Add**

### From a project

1. Open the project
2. Go to **Settings** or the project header
3. Click **Add to Program**
4. Select the target program

### When creating a project

During project creation, you can assign it to a program in the wizard.

## Viewing program projects

The program detail page displays all linked projects with:

- Project name and description
- Status indicator
- Timeline
- Stakeholder count
- Impact count

### Card view

Visual cards showing project status at a glance.

### List view

Table format with sortable columns and filtering.

## Removing projects

1. Open the program
2. Find the project
3. Click **Remove from Program**
4. Confirm removal

<Note>
  Removing a project from a program doesn't delete the project—it only unlinks them.
</Note>

## Project organization

### Status filtering

Filter program projects by status:

- All
- Active
- On-hold
- Completed

### Search

Search projects by name or description within the program.

## Cross-project insights

Programs enable analysis across projects:

### Shared stakeholders

Identify stakeholders affected by multiple projects in the program. These individuals may experience compounded change and need additional support.

### Overlapping impacts

Find change impacts that affect the same processes, systems, or teams across projects. Consider consolidating related impacts.

### Timeline conflicts

Visualize project timelines together to identify:

- Concurrent go-lives
- Resource conflicts
- Stakeholder overload periods

## Reporting across projects

Program-level metrics aggregate data:

- **Total stakeholders** - Unique people across all projects
- **Total impacts** - Combined change impacts
- **Average readiness** - Stakeholder readiness across the program
- **Completion rate** - Percentage of impacts addressed

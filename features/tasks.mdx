Leap Change includes a built-in task management system to help you track change management activities, action items, and follow-ups.

## Kanban board

Visualize your tasks in a kanban-style board with customizable columns.

### Default columns

| Column | Description |
|--------|-------------|
| Planned | Tasks identified but not yet started |
| Up next | Tasks ready to begin |
| In progress | Work currently underway |
| In review | Tasks completed and being reviewed |
| Done | Completed tasks |
| Canceled | Tasks no longer needed |

### Moving tasks

Drag and drop tasks between columns to update their status, or click on a task to edit its status directly.

## Creating tasks

1. Click **Add task** in any column
2. Enter task details:
   - **Title**: Brief description
   - **Description**: Additional details
   - **Assignee**: Who is responsible
   - **Project**: Link to an initiative
   - **Due date**: When it's needed
   - **Priority**: Urgency level
   - **Labels**: Tags for categorization
3. Click **Create**

## Task attributes

### Priority levels

- **High**: Urgent, needs immediate attention
- **Medium**: Important, standard timeline
- **Low**: Nice to have, flexible timing

### Labels

Create custom labels to categorize tasks:

- Communication
- Training
- Documentation
- Meeting
- Follow-up

### Due dates

Set deadlines to track timing:

- Overdue tasks are highlighted
- Filter to see upcoming deadlines
- Calendar view available

## Assignees

Assign tasks to team members:

1. Open a task
2. Select an assignee from organization members
3. The assignee receives notification (if enabled)

### My tasks

Filter to see only tasks assigned to you for focused work.

## Linking to projects

Connect tasks to specific initiatives:

- Filter tasks by project
- See all tasks related to an initiative
- Track progress per project

## Bulk operations

Manage multiple tasks at once:

1. Select tasks using checkboxes
2. Apply bulk actions:
   - Change status
   - Update assignee
   - Delete tasks

## Inbox

The **Inbox** provides a linear view of tasks:

- See all tasks in a list format
- Quick status updates
- Efficient for processing many items

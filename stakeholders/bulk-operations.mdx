title: Bulk operations
description: Update multiple stakeholders efficiently
---

Bulk operations let you make changes to multiple stakeholders at once, saving time when managing large groups.

## Selecting stakeholders

### Manual selection

1. Navigate to the **Stakeholders** table view
2. Click the checkbox next to each stakeholder you want to select
3. Or click the header checkbox to select all visible stakeholders

### Select with filters

1. Apply filters to narrow down stakeholders (department, project, location, etc.)
2. Select the filtered stakeholders
3. Perform bulk actions on the filtered set

## Available bulk actions

When stakeholders are selected, a bulk actions bar appears at the bottom of the screen.

### Bulk edit

Update fields for all selected stakeholders:

1. Click **Bulk Edit**
2. Choose the field to update:
   - **Project** - Assign to a project
   - **Department** - Update department
   - **Business unit** - Update business unit
   - **Location** - Update location
   - **Country** - Update country
   - **Influence level** - Set influence
   - **Support level** - Set support
   - **Readiness level** - Set readiness
3. Select or enter the new value
4. Click **Apply** to update all selected stakeholders

<Warning>
  Bulk edits cannot be undone. Review your selection carefully before applying changes.
</Warning>

### Add to group

Add selected stakeholders to one or more groups:

1. Click **Add to Group**
2. Select the target group(s)
3. Click **Add**

### Add to project

Assign selected stakeholders to a project:

1. Click **Add to Project**
2. Select the project
3. Click **Add**

### Send communication

Initiate a communication to selected stakeholders:

1. Click **Send Message**
2. Choose the channel (Email, Teams, Slack)
3. Compose and send the message

### Delete

Remove selected stakeholders:

1. Click **Delete**
2. Confirm the deletion

<Warning>
  Deleting stakeholders is permanent. Associated data (survey responses, activities) will also be deleted.
</Warning>

## Export

Export selected stakeholders to CSV:

1. Select stakeholders (or select all)
2. Click **Export**
3. Choose export options:
   - All fields or selected columns
   - Include project assignments
4. Download the CSV file

## Custom table views

Save filtered views for quick access to common stakeholder segments:

### Creating a view

1. Apply filters and column settings
2. Click **Save View**
3. Name the view
4. Choose visibility:
   - **Personal** - Only you can see it
   - **Shared** - All team members can access it
5. Save

### Using saved views

1. Click the view dropdown (shows current view name)
2. Select a saved view
3. The table updates with saved filters and columns

### Managing views

- **Edit** - Update filters or columns
- **Rename** - Change the view name
- **Delete** - Remove the view
- **Set as default** - Load this view when opening Stakeholders

## Best practices

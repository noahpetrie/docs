---
title: Stakeholder groups
description: Organize stakeholders into logical groups
---

Stakeholder groups help you organize and manage stakeholders with shared characteristics. Groups make it easier to send targeted communications, analyze readiness by segment, and apply bulk updates.

## Accessing groups

Navigate to **Stakeholders** → **Groups** tab to view and manage groups.

## Creating groups

1. Click **Create Group**
2. Enter group details:
   - **Name** - Descriptive group name
   - **Description** - Purpose or criteria for the group
   - **Color** - Visual identifier for the group
3. Add members:
   - Search for stakeholders by name
   - Select multiple stakeholders at once
   - Use filters to find stakeholders matching criteria
4. Click **Create**

## Group types

### Manual groups

Create groups by manually selecting members. Useful for:

- Cross-functional teams
- Project-specific groups
- Change champion networks
- Training cohorts

### Synced groups (Microsoft 365)

If Microsoft 365 integration is enabled with group sync:

- Entra ID groups automatically create corresponding stakeholder groups
- Membership stays synchronized
- Updates happen on each sync cycle

<Note>
  Synced groups show a "Synced from Microsoft 365" badge. Manual edits to membership will be overwritten on the next sync.
</Note>

## Managing groups

### Edit group

1. Click the group name to open it
2. Click **Edit Group**
3. Update name, description, or members
4. Save changes

### Add members

Multiple ways to add stakeholders to groups:

**From the group page:**
1. Open the group
2. Click **Add Members**
3. Search and select stakeholders

**From stakeholder profile:**
1. Open a stakeholder profile
2. Find the **Groups** section
3. Click **Add to Group**
4. Select target groups

**Using bulk actions:**
1. Select stakeholders in the table view
2. Click **Add to Group** in the bulk actions bar
3. Choose the target group

### Remove members

1. Open the group
2. Select members to remove
3. Click **Remove from Group**

### Delete group

1. Open the group
2. Click **Delete Group**
3. Confirm deletion

<Warning>
  Deleting a group only removes the grouping—stakeholders are not deleted.
</Warning>

## Using groups

### Communications

Target groups in communication activities:

1. Create a communication activity
2. In **Target Audience**, select **Group**
3. Choose the group to target

### Surveys

Send surveys to specific groups:

1. Create or open a survey
2. Click **Send Invitations**
3. Filter by group or select a group directly

### Filters

Filter the stakeholder table by group:

1. Open the **Filters** panel
2. Select **Group** filter
3. Choose one or more groups

### Reports

Analyze data by group:

- Filter reports by stakeholder group
- Compare readiness across groups
- Track engagement by segment

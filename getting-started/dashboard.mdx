The home dashboard provides an at-a-glance view of your change management activities and quick access to common tasks.

## Dashboard overview

When you sign in, the home screen displays:

### Greeting and date

A personalized greeting with the current date helps orient you at the start of your session.

### Key metrics

Quick statistics showing:

| Metric | Description |
|--------|-------------|
| **Stakeholders** | Total stakeholders in your organization |
| **Change impacts** | Number of tracked change impacts |
| **Surveys** | Active and completed surveys |
| **AI tools** | Available AI capabilities |

Click any metric card to navigate directly to that section.

### Quick actions

Access common tasks without navigating through menus:

- **Stakeholder management** - View and manage stakeholders
- **Change impacts** - Track and document impacts
- **Reports and analytics** - Generate insights
- **AI assistant** - Get AI-powered guidance
- **Surveys** - Create and manage surveys
- **Team** - Manage organization members

### Getting started checklist

For new users, a progress tracker guides you through initial setup:

1. **Add a stakeholder** - Create your first stakeholder record
2. **Record an impact** - Document a change impact
3. **Explore AI** - Try the AI assistant
4. **Generate a report** - View analytics

Completed steps are marked with a checkmark. The checklist disappears once you've completed the essentials.

## Navigation

From the home dashboard, you can navigate to any section:

### Sidebar navigation

The sidebar provides access to all major features:

- **Home** - Return to dashboard
- **Projects** - Manage initiatives
- **Stakeholders** - View stakeholder database
- **Change impacts** - Track impacts
- **Surveys** - Manage surveys
- **Communications** - Plan communications
- **Tasks** - Kanban board
- **Reports** - Analytics and visualizations
- **Settings** - Account and organization settings

### Quick search

Use the search bar or keyboard shortcut (`Cmd/Ctrl + K`) to:

- Find stakeholders by name
- Search for impacts
- Navigate to any section
- Access recent items

## Customization

The dashboard adapts to your usage:

- **Getting started** appears for new users
- **Quick actions** provide one-click access to frequent tasks
- **Metrics** reflect your organization's data

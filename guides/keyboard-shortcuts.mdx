Leap Change supports keyboard shortcuts to help you work more efficiently. Access the full list anytime by pressing `?` or `Cmd/Ctrl + K`.

## Command palette

The command palette provides quick access to any action:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |
| `Escape` | Close command palette |

From the command palette, start typing to search for:

- Pages and navigation
- Actions (create, edit, delete)
- Recent items
- Settings

## Global shortcuts

These shortcuts work from anywhere in the application:

| Shortcut | Action |
|----------|--------|
| `?` | Show keyboard shortcuts |
| `G then H` | Go to Home |
| `G then S` | Go to Stakeholders |
| `G then I` | Go to Change Impacts |
| `G then T` | Go to Tasks |
| `G then C` | Go to Communications |
| `G then R` | Go to Reports |
| `G then P` | Go to Projects |

## Creating items

Quick-create shortcuts:

| Shortcut | Action |
|----------|--------|
| `C then S` | Create stakeholder |
| `C then I` | Create impact |
| `C then T` | Create task |
| `C then P` | Create project |

## Navigation

Move around the interface:

| Shortcut | Action |
|----------|--------|
| `↑` / `↓` | Navigate list items |
| `Enter` | Select/open item |
| `Escape` | Close dialog/modal |
| `Tab` | Move to next field |
| `Shift + Tab` | Move to previous field |

## Table actions

When viewing tables (stakeholders, impacts, etc.):

| Shortcut | Action |
|----------|--------|
| `J` / `K` | Move down/up in list |
| `Space` | Select item |
| `Shift + Space` | Select range |
| `A` | Select all |
| `E` | Edit selected |
| `Delete` | Delete selected |

## Task board

On the Kanban board:

| Shortcut | Action |
|----------|--------|
| `←` / `→` | Move task to previous/next column |
| `N` | Create new task |
| `Enter` | Open task details |

## Settings

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + ,` | Open settings |

## Tips

<AccordionGroup>
  <Accordion title="Learn gradually">
    Start with the command palette (Cmd/Ctrl + K) and learn other shortcuts as you use the platform more.
  </Accordion>
  <Accordion title="Use sequences">
    Two-key shortcuts like "G then H" mean press G, release, then press H.
  </Accordion>
  <Accordion title="Customize if needed">
    Enterprise customers can request custom keyboard shortcut configurations.
  </Accordion>
</AccordionGroup>

Leap Change supports keyboard shortcuts for faster navigation and common actions.

## Viewing shortcuts

Press `?` to open the keyboard shortcuts modal from anywhere in the application.

## Navigation shortcuts

Quickly jump to different sections using `G` followed by a second key:

| Shortcut | Action |
|----------|--------|
| `G` then `H` | Go to Home |
| `G` then `S` | Go to Stakeholders |
| `G` then `P` | Go to Projects |
| `G` then `C` | Go to Change Impacts |
| `G` then `V` | Go to Surveys |
| `G` then `K` | Go to Kanban board |
| `G` then `R` | Go to Reports |
| `G` then `T` | Go to Settings |

<Tip>
  Press `G` and release, then press the second key. You don't need to hold both keys simultaneously.
</Tip>

## Global shortcuts

These shortcuts work from anywhere in the application:

| Shortcut | Action |
|----------|--------|
| `⌘/Ctrl` + `K` | Open command palette |
| `⌘/Ctrl` + `.` | Toggle sidebar |
| `N` | Create new item (context-aware) |
| `Esc` | Close modals and dialogs |
| `?` | Show keyboard shortcuts |

## Command palette

The command palette (`⌘/Ctrl` + `K`) provides quick access to:

- **Navigation** - Jump to any page
- **Search** - Find stakeholders, impacts, surveys
- **Actions** - Create new items, toggle settings
- **Theme** - Switch between light, dark, and system themes

### Using the command palette

1. Press `⌘/Ctrl` + `K` to open
2. Start typing to search
3. Use arrow keys to navigate results
4. Press `Enter` to select
5. Press `Esc` to close

## Context-aware creation

Pressing `N` creates a new item based on your current location:

| Current page | Creates |
|--------------|--------|
| Stakeholders | New stakeholder |
| Change Impacts | New impact |
| Surveys | New survey |
| Tasks/Kanban | New task |
| Communications | New communication plan |

## Notes on shortcuts

<Note>
  Shortcuts are disabled when you're typing in input fields, text areas, or when a modal is open (unless the shortcut explicitly supports modals).
</Note>

### Platform differences

- **macOS**: Uses `⌘` (Command) for modifier shortcuts
- **Windows/Linux**: Uses `Ctrl` for modifier shortcuts

The shortcuts modal automatically shows the correct modifier for your platform.

Leap Change includes a command palette and keyboard shortcuts to help power users work more efficiently.

## Command palette

Open the command palette to quickly access any feature:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open command palette |

### Available commands

From the command palette, you can:

**Quick add**
- Create a new change impact
- Add a stakeholder
- Create a survey
- Start a communication plan

**Navigation**
- Jump to any section
- Open recent items
- Search across your data

**Actions**
- Run reports
- Export data
- Access settings

## Global shortcuts

These shortcuts work from anywhere in the application:

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Command palette |
| `?` | Show keyboard shortcuts help |
| `Esc` | Close modal or dialog |

## Navigation shortcuts

Quickly navigate between sections:

| Shortcut | Destination |
|----------|-------------|
| `G` then `H` | Go to Home |
| `G` then `P` | Go to Projects |
| `G` then `S` | Go to Stakeholders |
| `G` then `I` | Go to Impacts |
| `G` then `R` | Go to Reports |

## Table shortcuts

When working with data tables:

| Shortcut | Action |
|----------|--------|
| `↑` `↓` | Navigate rows |
| `Enter` | Open selected item |
| `Space` | Toggle row selection |
| `/` | Focus search |

## Kanban board shortcuts

When using the task board:

| Shortcut | Action |
|----------|--------|
| `N` | New task |
| `←` `→` | Move task between columns |
| `Enter` | Open task details |

## Tips

<Tip>
  Press `Cmd/Ctrl + K` and start typing to search for anything—stakeholders, impacts, projects, or commands.
</Tip>

<Tip>
  The command palette remembers your recent actions, making repeated tasks even faster.
</Tip>

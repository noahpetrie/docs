---
title: Bulk operations
description: Update multiple stakeholders at once
---

Bulk operations let you efficiently update many stakeholders simultaneously.

## Selecting stakeholders

1. Go to **Stakeholders** table view
2. Use checkboxes to select stakeholders:
   - Click individual checkboxes
   - Click the header checkbox to select all visible
   - Apply filters first to narrow your selection

## Available operations

With stakeholders selected, click **Bulk Actions** to access:

### Edit fields

Update common fields across selected stakeholders:

- Influence level
- Support level
- Readiness level
- Department
- Location

### Assign to project

Link selected stakeholders to a project:

1. Click **Assign to Project**
2. Select the target project
3. Confirm the assignment

### Add to group

Add stakeholders to a group:

1. Click **Add to Group**
2. Select the target group
3. Confirm

### Delete

Remove selected stakeholders:

1. Click **Delete**
2. Review the list
3. Confirm deletion

<Warning>
  Deleting stakeholders removes them from all projects, surveys, and historical data. This action cannot be undone.
</Warning>

## Export

Export stakeholder data for external use:

1. Select stakeholders (or select all)
2. Click **Export**
3. Choose format (CSV)
4. Download the file

Exports include all stakeholder fields and group memberships.

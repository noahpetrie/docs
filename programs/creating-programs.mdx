---
title: Creating programs
description: Set up programs to group related change initiatives
---

Programs help you organize related change initiatives for coordinated management and reporting.

## Create a new program

1. Navigate to **Programs** in the sidebar
2. Click **New program**
3. Fill in the program details
4. Click **Create**

## Program details

### Required fields

| Field | Description |
|-------|-------------|
| **Title** | Clear, descriptive program name |
| **Description** | Overview of objectives and scope |
| **Change owner** | Person accountable for delivery |
| **Sponsor** | Executive sponsor |
| **Departments** | Business units involved |
| **Go-live date** | Target completion date |

### Optional fields

| Field | Description |
|-------|-------------|
| **Planned start date** | When work begins |
| **Category** | Type of change |
| **Geographic scope** | Regions affected |
| **Impact severity** | Overall impact level |
| **Audience size** | Number of people affected |
| **Dependencies** | Prerequisites or external factors |
| **Tags** | Labels for organization |

## Best practices

<Tip>
  Use consistent naming conventions across programs to make navigation easier. For example: "[Year] - [Category] - [Name]"
</Tip>

### Program scope

- Group initiatives with shared objectives
- Keep programs focused (5-10 initiatives max)
- Ensure clear ownership boundaries

### Timeline planning

- Set realistic go-live dates
- Account for dependencies between initiatives
- Build in buffer time for unexpected delays

### Stakeholder considerations

- Identify overlapping stakeholder groups early
- Plan for cumulative change impact
- Coordinate communication timing

## Editing programs

1. Open the program from the list
2. Click the edit icon
3. Update any fields
4. Save changes

## Deleting programs

<Warning>
  Deleting a program does not delete its initiatives. Initiatives will become unlinked from any program.
</Warning>

1. Open the program
2. Click **Delete**
3. Confirm the deletion

## Access control

Programs follow the same access model as initiatives:

- **Organization members** can view programs
- **Admins** can create, edit, and delete programs

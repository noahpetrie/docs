## Integration benefits

### Automatic stakeholder sync

Keep your stakeholder data current:

- Import users from your identity provider
- Sync organizational structure and groups
- Update contact information automatically
- Maintain accurate department and role data

### Direct communication

Reach stakeholders where they work:

- Send Teams messages from stakeholder profiles
- Post to Slack channels for updates
- Track communication delivery
- Support survey distribution

### Channel monitoring

Stay informed about change-related discussions:

- Monitor Teams channels for mentions
- Capture insights for impact analysis
- Track stakeholder sentiment

## Managing integrations

1. Go to **Settings** → **Integrations**
2. View all available integrations
3. Connect or disconnect as needed
4. Configure integration settings

<Note>
  Integration management requires admin role.
</Note>

## Data security

Integrations follow security best practices:

- OAuth 2.0 authentication
- Minimal required permissions
- Encrypted data transfer
- Revocable access tokens

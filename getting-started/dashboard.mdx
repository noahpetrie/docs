The Dashboard is your central hub in Leap Change, providing at-a-glance visibility into your change initiatives and quick access to common actions.

## Home dashboard

When you sign in, you land on the Home dashboard. This view shows:

### Key metrics

- **Active projects** - Number of ongoing change initiatives
- **Total stakeholders** - People tracked across all projects
- **Open impacts** - Change impacts requiring attention
- **Pending surveys** - Surveys awaiting responses

### Quick actions

Start common tasks directly from the dashboard:

- **Create project** - Start a new change initiative
- **Add stakeholder** - Add someone to your stakeholder database
- **New impact** - Document a change impact
- **Send survey** - Distribute a readiness survey

### Recent activity

See what's happening across your organization:

- Recently created or updated projects
- New stakeholders added
- Survey responses received
- Impacts marked complete

## Navigation sidebar

The sidebar provides access to all Leap Change features:

| Section | Description |
|---------|-------------|
| **Home** | Return to the main dashboard |
| **Projects** | Manage change initiatives and programs |
| **Stakeholders** | View and manage stakeholder database |
| **Change Impacts** | Track and mitigate change impacts |
| **Surveys** | Create surveys and view responses |
| **Communications** | Plan and track communications |
| **Tasks** | Kanban board for work items |
| **Reports** | Analytics and visualizations |
| **AskLeap** | AI-powered assistant |
| **Settings** | Account, organization, and integrations |

## Organization dashboard

For organization-wide visibility:

1. Click **Dashboard** in the sidebar
2. Select **Org** tab
3. View aggregate metrics across all initiatives

The organization dashboard shows:

- **Team activity** - Actions by organization members
- **Project status** - Overview of all projects by status
- **Impact summary** - Total impacts by severity
- **Readiness trends** - Stakeholder readiness over time

## Customizing your view

### Filters

Apply filters to focus the dashboard:

- By project - See data for specific initiatives
- By date range - Focus on recent activity
- By status - Show only active or completed items

### Saved views

Save commonly used filter combinations:

1. Apply your desired filters
2. Click **Save view**
3. Name your saved view
4. Access it from the view dropdown

## Notifications

Stay informed about important updates:

- **Bell icon** - Click to see recent notifications
- **Unread count** - Badge shows pending notifications
- **Mark as read** - Clear individual or all notifications

### Notification types

- Survey responses received
- Tasks assigned to you
- Upcoming activity deadlines
- Impact status changes
- Team member actions

Configure notification preferences in **Settings** → **Notifications**.

## Keyboard shortcuts

Navigate efficiently with keyboard shortcuts:

| Shortcut | Action |
|----------|--------|
| `G` then `H` | Go to Home |
| `G` then `P` | Go to Projects |
| `G` then `S` | Go to Stakeholders |
| `G` then `I` | Go to Impacts |
| `G` then `T` | Go to Tasks |
| `/` | Open search |
| `?` | Show all shortcuts |

<Tip>
  Press `?` anywhere in Leap Change to see all available keyboard shortcuts for the current page.
</Tip>

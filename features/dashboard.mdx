The Dashboard provides a centralized view of your change management activities, helping you quickly understand the status of your initiatives and identify areas needing attention.

## Home dashboard

Access the home dashboard by clicking **Home** in the sidebar.

### Key metrics

The dashboard displays at-a-glance metrics:

| Metric | Description |
|--------|-------------|
| **Active initiatives** | Number of ongoing change projects |
| **Total stakeholders** | People tracked across all initiatives |
| **Open impacts** | Change impacts not yet resolved |
| **Upcoming communications** | Scheduled activities in the next 7 days |

### Quick actions

Common actions are accessible from the dashboard:

- Create a new initiative
- Add a stakeholder
- Log a change impact
- Start a survey

### Recent activity

See what's happened recently:

- New stakeholders added
- Impact status changes
- Survey responses received
- Communications completed

## Organization dashboard

For organization-wide visibility, navigate to **Dashboard** > **Org**.

### Cross-initiative view

Compare metrics across all initiatives:

- Initiative health indicators
- Stakeholder distribution
- Impact severity breakdown
- Communication coverage

### Team activity

See what team members have been working on:

- Recent changes by team member
- Active assignments
- Pending items requiring attention

## Filtering dashboard data

Customize what you see:

### By initiative

1. Click the initiative filter
2. Select one or more initiatives
3. Metrics update to show selected scope

### By date range

1. Click the date picker
2. Select a custom range
3. View data for that period

## Dashboard widgets

The dashboard includes these visual components:

### Impact severity chart

Pie chart showing the distribution of impacts by severity level (Critical, High, Medium, Low).

### Readiness breakdown

Shows stakeholder readiness distribution:

- Champions
- Ready
- Aware
- Resistant

### Timeline

Visual representation of:

- Upcoming milestones
- Scheduled communications
- Due dates for impacts

### Recent surveys

Quick view of:

- Open surveys
- Response rates
- Recent completions

## Personalizing your dashboard

### Favorites

Mark initiatives as favorites for quick access:

1. Open an initiative
2. Click the star icon
3. Favorite initiatives appear prominently on your dashboard

### Notifications

Configure what appears in your activity feed:

1. Go to **Settings** > **Notifications**
2. Select which activities to track
3. Save your preferences

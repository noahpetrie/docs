---
title: Tracking communications
description: Monitor communication plan progress and completion
---

Track your communication plan execution to ensure stakeholders receive timely, complete information about your changes.

## Plan dashboard

Each communication plan has a dashboard showing:

- **Completion rate** - Percentage of activities completed
- **Upcoming activities** - What's scheduled next
- **Overdue activities** - Missed deadlines needing attention
- **Timeline progress** - Visual representation of plan progress

## Activity tracking

### Status management

Update activity status as you execute:

1. Open the activity
2. Click **Mark Complete**
3. Optionally add completion notes

Bulk updates:
1. Select multiple activities
2. Click **Mark Selected Complete**

### Overdue handling

When an activity passes its scheduled date without completion:

- Marked as **Overdue** in red
- Appears in the plan dashboard
- Notifications sent to owner (if enabled)

For overdue items:
- **Complete** - Mark done with the actual completion date
- **Reschedule** - Move to a new date
- **Cancel** - Remove from plan if no longer needed

## Coverage tracking

### Stakeholder coverage

See which stakeholders have been reached:

- **Reached** - Included in at least one completed activity
- **Pending** - Included in upcoming activities
- **Not covered** - Not in any planned activities

Access this in **Plan** → **Coverage** tab.

### Impact coverage

Track communications per change impact:

- Which impacts have associated communications
- Gaps where impacts need communication activities
- Links from impacts to completed activities

## Reports and exports

### Export plan

Download your communication plan:

- **PDF** - Formatted timeline and activity list
- **CSV** - Raw data for external analysis
- **Calendar** - ICS file for calendar apps

### Activity log

View history of all plan changes:

- Activities added, modified, deleted
- Status changes
- Who made changes and when

## Notifications

Configure notifications in plan settings:

| Event | Notification |
|-------|--------------|
| Activity due tomorrow | Reminder to owner |
| Activity overdue | Alert to owner and plan manager |
| Plan milestone reached | Update to stakeholders |
| All activities completed | Completion notification |

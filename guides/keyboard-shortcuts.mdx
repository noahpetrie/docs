Leap Change supports keyboard shortcuts to help you work more efficiently without leaving the keyboard.

## Command palette

The command palette is your fastest way to navigate:

| Shortcut | Action |
|----------|--------|
| **Cmd+K** (Mac) / **Ctrl+K** (Windows) | Open command palette |

From the command palette, you can:

- Search and navigate to any page
- Find stakeholders by name
- Find projects by name
- Execute quick actions
- Switch themes

## Global shortcuts

Available anywhere in the application:

| Shortcut | Action |
|----------|--------|
| **Cmd+K** / **Ctrl+K** | Open command palette |
| **Cmd+/** / **Ctrl+/** | Toggle sidebar |
| **Esc** | Close dialogs and modals |

## Navigation shortcuts

Quick access to main sections via command palette:

1. Press **Cmd+K** / **Ctrl+K**
2. Type the section name:
   - "home" - Go to dashboard
   - "stakeholders" - View stakeholders
   - "impacts" - View change impacts
   - "surveys" - View surveys
   - "tasks" - Open kanban board
   - "reports" - View reports
   - "settings" - Open settings

## Quick actions

Execute common actions from the command palette:

| Action | How to access |
|--------|---------------|
| Add stakeholder | Cmd+K → "add stakeholder" |
| Add impact | Cmd+K → "add impact" |
| Add task | Cmd+K → "add task" |
| Add survey | Cmd+K → "add survey" |

## Theme switching

Change the theme quickly:

1. Open command palette (**Cmd+K**)
2. Type "theme"
3. Select Light, Dark, or System

## Search shortcuts

The command palette includes real-time search:

- **Stakeholders**: Type any part of a stakeholder's name
- **Projects**: Type any part of a project name
- Results appear instantly as you type

## Accessibility

Keyboard shortcuts are designed for accessibility:

- All major features are keyboard-accessible
- Focus indicators show your current position
- Tab navigation works throughout the interface
- Screen reader compatible

The Stakeholder Map provides a geographic view of your stakeholders, helping you understand distribution and plan location-specific communications.

## Map features

### Location markers

Stakeholders appear as markers on the map:

- **Marker size** - Indicates number of stakeholders at that location
- **Marker color** - Can represent readiness level, influence, or other attributes
- **Clusters** - Groups nearby stakeholders when zoomed out

### Map controls

- **Zoom** - Focus on regions or specific locations
- **Pan** - Move around the map
- **Layers** - Toggle different data overlays

## Using the map

### View by attribute

Color-code markers by:

- **Readiness level** - Champion, Ready, Aware, Resistant
- **Influence** - High, Medium, Low
- **Department** - By organizational unit
- **Project assignment** - By change initiative

### Click interactions

Click a marker or cluster to:

- See stakeholder count
- View names and details
- Navigate to stakeholder profiles
- Filter table to that location

### Regional analysis

Identify patterns:

- Which regions have the most stakeholders?
- Are certain regions more resistant?
- Where should in-person communications focus?
- Which locations need local champions?

## AI-powered analysis

The Stakeholder Map includes AI analysis features. Click **Analyze** to get:

### Insights

- Distribution patterns across regions
- Concentration areas requiring attention
- Coverage gaps in your stakeholder data

### Relationships

- Cross-regional connections
- Influence networks
- Department distributions by location

### Strategy recommendations

- Suggested engagement approaches by region
- Communication channel recommendations
- Local champion identification

### Risk identification

- Regions with high resistance
- Areas with low engagement
- Locations lacking representation

## Data requirements

For stakeholders to appear on the map:

- **Location field** - City, office, or address
- **Country** - For international distribution

Leap Change geocodes locations automatically. If a location can't be found, it won't appear on the map.

### Improving geocoding

If stakeholders aren't appearing:

1. Check the location field is populated
2. Use standard location formats (City, Country)
3. Avoid abbreviations when possible
4. Verify country codes are correct

## Export

- **Image** - PNG of the current map view
- **Location data** - CSV with coordinates
- **Report** - PDF with map and regional breakdown

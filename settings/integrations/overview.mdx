Leap Change integrates with external platforms to streamline your workflow, sync stakeholder data, and enable cross-platform communication.

## Accessing integrations

1. Go to **Settings** in the sidebar
2. Click **Integrations**
3. Browse available integrations
4. Click on an integration to configure it

## Integration categories

### Identity & Directory

Sync users and groups from identity providers:

- Import stakeholders automatically
- Keep data synchronized
- Match organizational structure

### Communication

Send messages and notifications through external platforms:

- Direct stakeholder messages
- Survey invitations
- Change announcements

## Admin requirements

<Warning>
  Only organization **admins** can configure integrations. Some integrations also require admin permissions in the external service.
</Warning>

## Connection status

Each integration shows its connection status:

| Status | Description |
|--------|-------------|
| **Connected** | Integration is active and working |
| **Not connected** | Integration available but not configured |
| **Error** | Connection issue requiring attention |

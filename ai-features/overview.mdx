Leap Change integrates AI capabilities to help you work faster and surface insights you might otherwise miss. These features are powered by AskLeap, the AI assistant built into the platform.

## Available AI features

### Transcript impact extraction

Upload meeting transcripts to automatically identify change impacts:

- Supported formats: VTT, DOCX, TXT, PDF
- AI extracts potential impacts from discussions
- Review and approve suggestions
- Save hours of manual analysis

[Learn more about AI extraction](/change-impacts/ai-extraction)

### AI chat assistant

Chat with AskLeap for change management guidance:

- Ask questions about best practices
- Get recommendations for your situation
- Explore change management methodologies
- Draft communication messages

### Impact insights

Get AI-generated insights on individual impacts:

- Suggested mitigation strategies
- Risk analysis
- Stakeholder recommendations
- Communication approaches

### Stakeholder analysis

AI-powered stakeholder intelligence:

- Identify key influencers
- Suggest engagement strategies
- Highlight potential advocates or resistors
- Recommend communication approaches

## Accessing AI features

### From the sidebar

Navigate to **AI Features** to access:

- Chat interface
- Transcript upload
- Feature documentation

### In context

AI features appear throughout the app:

- **Impact detail pages** - Insights panel
- **Stakeholder profiles** - Analysis suggestions
- **Import workflows** - Transcript extraction

## Usage and quotas

AI features consume usage quota based on your plan:

| Plan | Monthly quota |
|------|---------------|
| Basic | 100 requests |
| Pro | 1,000 requests |
| Enterprise | Unlimited |

### Tracking usage

Monitor your usage in **Settings** → **AskLeap Usage**:

- Requests used this month
- Quota remaining
- Usage by feature type
- Historical usage

### Quota exceeded

When quota is reached:

- AI features become unavailable
- Historical results remain accessible
- Usage resets on billing cycle
- Upgrade for more capacity

## Data privacy

AI processing follows these principles:

- Your data is not used to train AI models
- Conversations are not stored beyond your session
- Processing happens on secure infrastructure
- Enterprise customers can use their own AI endpoints

<Tip>
  Start with transcript extraction—it typically provides the highest time savings for change practitioners.
</Tip>
